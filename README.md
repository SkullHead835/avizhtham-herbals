# 🌿 Avizhtham Herbal Website

A modern, responsive website built for **Avizhtham Herbal** using **Next.js**, **TypeScript**, and modern frontend technologies. The project is designed to provide a clean, fast, and user-friendly experience while showcasing the brand, products, and services.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/avizhtham-herbal-website.git
```

Navigate into the project

```bash
cd avizhtham-herbal-website
```

Install dependencies

Using npm

```bash
npm install
```

or using pnpm

```bash
pnpm install
```

---

## ▶️ Run the Development Server

Using npm

```bash
npm run dev
```

Using pnpm

```bash
pnpm dev
```

Open your browser and visit

```
http://localhost:3000
```

---

## 🏗️ Build for Production

```bash
npm run build
```

or

```bash
pnpm build
```

Start the production server

```bash
npm start
```

or

```bash
pnpm start
```
